# 常用命令

1. 启动命令

python manage.py runserver

1. 生成迁移

python manage.py makemigrations goods_detection_server

2. 查看迁移SQL

python manage.py sqlmigrate goods_detection_server 0001

3. 执行迁移

python manage.py migrate
